# Capstone Project — maximus dinings (Food Delivery App)

## Project Theme
**Food Delivery App** — a responsive front-end landing/catalog page built with HTML5 & CSS3 (Flexbox + Grid).

---

## Files
- `index.html` — Semantic HTML structure containing header, hero, filters, menu, restaurants, offers, contact form, and cart summary (UI only).
- `style.css` — Complete responsive CSS with variables, hover effects, transitions, and a keyframe animation.
- `images/` — Add images listed below.

---

## Requirements implemented
- Semantic HTML5 elements: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`.
- Responsive layout using **Flexbox** (header, hero) and **CSS Grid** (menu, cards).
- At least two breakpoints: **Desktop ≥1024px**, **Tablet ≤768px**, **Mobile ≤480px**.
- Forms have `<label>` and required attributes.
- Accessibility touches: `aria-label`, `role`, `visually-hidden` helper for search label.
- Visual polish: CSS variables, hover effects, transitions, keyframe animation on hero image.
